# Mouhuba

Projet de bot Telegram et serveur VPN sur Ubuntu.

## Contenu
- **bot/** : Bot Telegram
- **vpn/** : Scripts de génération OpenVPN
- **config/** : Fichiers de configuration

## Installation
```bash
git clone https://github.com/mouhubaboyagoro-byte/Mouhuba.git
cd Mouhuba
python3 -m venv venv
source venv/bin/activate
pip install pyTelegramBotAPI
```

## Utilisation
```bash
source venv/bin/activate
python bot/bot.py
```
